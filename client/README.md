# PasteBox Client

React + Vite frontend for the PasteBox file-sharing platform.

## Run

```bash
npm install
npm run dev
```

Set `VITE_API_URL=https://your-render-api.example.com/api` in `.env`.

For local development, set `VITE_API_URL=http://localhost:6600/api`.

See the root [README](../README.md) for full project setup.
